Couple Ways is a dynamic web application designed to provide a way to plan and organize our journeys. This project showcases my skills in full-stack development, featuring a user-friendly interface with dark mode capabilities, responsive design, and robust trip management functionalities.

From the intuitive navigation bar to the seamless trip addition feature, Couple Ways offers a comprehensive solution for users to effortlessly plan and customize their travel experiences. Dive into the details of each trip with the responsive page design, incorporating features such as pricing information, photo uploads, YouTube video integration, and even a savings plans calculator for financial planning.

The project goes beyond basic functionality with a simulated trip budget feature and user authentication for personalized trip management. Whether you're a solo adventurer or planning with a partner, Couple Ways ensures a personalized and secure journey from start to finish.
